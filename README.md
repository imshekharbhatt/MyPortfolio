## 🚀 Quick Start

```bash
# Clone repository
git clone https://github.com/imshekharbhatt/MyPortfolio.git

# Navigate to project
cd MyPortfolio

# Open in default browser (No build process required!)
open index.html

# Or use live server for development
npx live-server --port=3000 --browser=chrome
```
